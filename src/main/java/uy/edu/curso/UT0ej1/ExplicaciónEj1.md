CONCEPTOS:

JVM: La máquina virtual Java, carga, verifica y ejecuta el código de bytes de Java. Se conoce como el intérprete o el núcleo del lenguaje de programación Java porque ejecuta la programación Java.

JRE: Java Runtime Environment, es un conjunto de herramientas de software responsables de la ejecución de Java en su sistema. utiliza espacio de almacenamiento dinámico para la asignación dinámica de memoria para objetos Java. JRE también se utiliza en JDB (Java Debugging).

JDK: Java Development Kit, es un kit de desarrollo de software que es un superconjunto de JRE. Es el componente fundacional que permite el desarrollo de aplicaciones Java y subprogramas Java. Es específico de la plataforma, por lo que se necesitan instaladores independientes para cada sistema operativo (por ejemplo, Mac, Unix y Windows).
